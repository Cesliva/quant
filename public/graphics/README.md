# Graphics Assets

This folder contains graphics assets for the Quant AI website.

## Folder Structure

- `logos/` - Company logos and brand assets
  - `logo.png` or `logo.svg` - Main logo
  - `logo-dark.png` or `logo-dark.svg` - Dark variant (if needed)
  - `logo-light.png` or `logo-light.svg` - Light variant (if needed)
  - `favicon.ico` - Favicon file
  - `favicon-16x16.png` - 16x16 favicon
  - `favicon-32x32.png` - 32x32 favicon
  - `apple-touch-icon.png` - Apple touch icon (180x180)

- `screenshots/` - Screenshot images for the landing page
  - Add your screenshot files here
  - Recommended formats: PNG, WebP, or optimized JPG
  - Recommended sizes: 1920x1080 or similar for hero images

## Usage

After adding files, reference them in your code like:
- `/graphics/logos/logo.png`
- `/graphics/screenshots/dashboard-screenshot.png`

## File Naming Convention

- Use lowercase with hyphens: `dashboard-overview.png`
- Be descriptive: `estimating-grid-view.png` not `img1.png`
- Include size if multiple variants: `logo-200x200.png`, `logo-400x400.png`
